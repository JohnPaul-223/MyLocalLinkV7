# LocaLink - Simple Live Tracker

A simple, real-time location tracking web app built with Vanilla HTML, CSS, and JavaScript.

## 🚀 Features
- **Real-time Map**: Live location sharing using Leaflet.
- **Supabase Integration**: Data persistence and Auth out of the box.
- **Minimalist Design**: Clean, easy-to-use mobile-responsive UI.

## 📦 How to Use
1. **Upload**: Simply upload `index.html`, `style.css`, and `script.js` to your GitHub.
2. **Deploy**: Connect your GitHub to Vercel. Since there are no build steps, it will go live instantly.
3. **Database**: Make sure you have run the `supabase_schema.sql` (from previous artifact) in your Supabase SQL Editor.

## 🛠️ Customization
To change your database, find the `SUPABASE_URL` and `SUPABASE_KEY` at the top of `script.js`.
